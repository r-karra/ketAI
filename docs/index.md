# ketAI

Welcome to the ketAI project site.

This repository contains a Vite-powered React application in the repository root. The GitHub Pages site uses the `docs/` folder as its source, so this page is a lightweight project landing page.

## About

ketAI is a sample AI Studio app built with React, Vite, and Tailwind-inspired tooling. It integrates with Gemini and provides a local development workflow.

## Local development

1. Install dependencies:
   `npm install`
2. Set your Gemini API key in `.env.local`:
   `GEMINI_API_KEY=<your-key>`
3. Start the app:
   `npm run dev`

## Notes

- The live Pages site is available at `https://r-karra.github.io/ketAI/`
- The application itself is served from the repository root during development via Vite.
